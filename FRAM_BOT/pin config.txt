Arduino UNO Pin Configuration
Bluetooth Module (HC-05 / HC-06)
TX (Bluetooth) → D3 (Arduino RX)

RX (Bluetooth) → D2 (Arduino TX) via voltage divider (to drop Arduino 5V to ~3.3V)

VCC → 5V

GND → GND

Motor Driver (L298N)
IN1 → D8

IN2 → D9

IN3 → D10

IN4 → D11

ENA → D5 (PWM)

ENB → D6 (PWM)

VCC (Motor Power) → Battery + (e.g., 7.4V–12V depending on motor specs)

GND → Arduino GND + Battery GND

Motor Outputs:

OUT1 & OUT2 → Left motors

OUT3 & OUT4 → Right motors




 RED (5V)  --> servo VCC (all servos red)
   BLACK (GND) --> servo GND (all servos black)
   Connect BLACK to Arduino GND
Servo signals:
   servo1 signal (yellow) -> D4
   servo2 signal (yellow) -> A1
   servo3 signal (yellow) -> A2

Servo Motors
Servo1 (Left/Right): Signal → D4, VCC → 5V, GND → GND

Servo2 (Up/Down): Signal → A1, VCC → 5V, GND → GND

Servo3 (Pick/Release): Signal → A2, VCC → 5V, GND → GND

Ultrasonic Sensor (HC-SR04) (if used)
Trig → D12

Echo → D13

VCC → 5V

GND → GND

Buzzer (for alerts)
Signal → A0

VCC → 5V

GND → GND
